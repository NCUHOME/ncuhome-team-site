# Ncuhome-team-site

:tada: 全新的家园工作室主站

## 技术栈

- React (with Hooks) & react-router-dom
- Sass
- Vite
- TypeScript
- SSR

## 开发指南

**安装依赖**

```shell
$ pnpm i
```

**本地调试**

```shell
$ pnpm dev
```

**作为 production 在 local 部署**

```shell
$ pnpm start
```
